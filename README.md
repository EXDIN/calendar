# Name - Calendar
  Calendar app with drag and drop

## Description
  Technologies
  - Vite
  - TypeScript
  - React
  - React-Router-Dom
  - RTK
  - React-beautiful-dnd
  - CSS in JS - stitches
  - Lucide-react

### URL 
    https://exdin.github.io/calendar/

### Usage
  Install:
    git clone https://github.com/EXDIN/calendar
    cd calendar
    npm install

  Run:
    npm run dev
